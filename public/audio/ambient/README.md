# Ambient Audio Files

Place MP3 or OGG loops here to replace the synthesized ambient soundscapes.
When a file is present, it takes priority over the procedural fallback.
The loader tries `.mp3` first, then `.ogg`.

## Requirements

- **Format:** MP3 or OGG (128–192 kbps recommended)
- **Duration:** 30–90 seconds, seamlessly loopable
- **Volume:** Normalize to -18 to -14 LUFS (these play as background ambience)
- **License:** CC0 / Public Domain only
